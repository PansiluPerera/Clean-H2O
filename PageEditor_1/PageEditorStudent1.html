Splash Screen

Created the splash screen using two seperate screens rather than including the website title and team members on the same page for increased readability and visuals. The ripple on the first screen is directly created using HTML and CSS rather than using GIFs. The team members splash screen shows a loading animation as well before redirecting to the main page of the website.

Shop

Created the shop page, incorperated with a seperate checkout page as well. Graphically designed each product using Ash and created a cart so that users can add or remove items as they please. An array method was used to collect the products added including dataset. tag to get name and price. Once user clicks checkout, the checkout page will load, asking for delivery and payment details. After filling each field, user can place the order which will redirect the user back to the shop page, zeroing the array, hence the cart. Also, user cannot proceed to the checkout page without adding any items.

"Access to Basic Sanitation Needs" Content Page

Researched related to this topic and included the most significant information from several sources. SDG 6 was chosen for this project because of its importance to the developing countries in this world and even areas in our own country as well, where clean water is a scarcity. Moderate CSS stylings were used to enhance the visibility of the information and give a more visually appealing look to the user.
